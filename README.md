## 💎 Technical Breakdown (What I applied)

* **CSS-Based Menu:** I designed a functional menu using CSS `checkbox` logic as a core. While the foundation is pure CSS, I integrated a small amount of **JavaScript** to handle menu closing more smoothly.
* **Layout & Layering:** I practiced using `z-index` layers and CSS masks (`mask-image`) to create gradient effects directly on images without needing external editing.
* **3D Interactions:** I implemented Z-axis transforms on the product cards to understand perspectives and pseudo-elements (`::before`, `::after`).
* **Fluidity:** Focused on transitions using `transform` and `scale` for GPU-accelerated performance.
* **HTML5 & CSS3:** Clean, semantic markup using **Flexbox** and **CSS Grid**.
